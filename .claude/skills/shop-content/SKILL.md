# Shop Content Skill

## Purpose
Max's business generates raw material daily (cuts, transformations, client reactions, view counts). This skill turns that raw material into two outputs: clean data in the repo, and content in his brand voice. Keep both honest and fast — he might be doing this between clients.

## Brand voice
- Confident, playful, young. Short sentences. No corporate-speak, no hashtag spam (3-5 targeted hashtags max).
- Hooks lead with transformation or curiosity ("He asked for 'just a trim'..."), never with "Check out my...".
- Self-aware about being a teenager with a garage shop — that's the charm, lean into it, never apologize for it.

## When logging an appointment
1. Append to the data file using the existing schema exactly. Date, service, price, new/returning, first name or initials only, tip.
2. Confirm back what was logged in one line.
3. If the entry makes a milestone (100th cut of the year, best month ever, new repeat-rate high), say so — milestones are content.

## When adding gallery content
1. Verify with Max that the client gave permission for the photo. If he hesitates at all, don't add it.
2. Optimize images for mobile (compress, correct dimensions per the gallery component).
3. Suggest a one-line caption for the pair in brand voice.

## When writing captions or post ideas
1. Ask what actually happened during the cut — the specific moment (client's reaction, a save, a risky bleach) is the content. Generic captions are banned.
2. Offer 2-3 options with different angles: transformation, humor, process.
3. Always end with where the link-in-bio points (booking page) when it fits naturally. Never force it.
4. If Max shares performance numbers from a past post, note the pattern (hook style, format, posting time) in a running `content-notes.md` file in the repo — this becomes his Viral Content Lab dataset later.

## Hard rules
- Real numbers only. Never round up views, cuts, or revenue.
- Client privacy: first names/initials, permission for photos, no location specifics.
- School-appropriate content only — admissions officers will see this.
